影像法 (Method of Images)

用虛擬電荷代替導體表面的感應電荷分布。

範例：點電荷 qqq 距接地導體平面 ddd → 用 −q-q−q 置於平面另一側 ddd 處替代。空間中任意點的電位：

V(x,y,z)=kqx2+y2+(z−d)2−kqx2+y2+(z+d)2V(x,y,z) = \frac{kq}{\sqrt{x^2+y^2+(z-d)^2}} - \frac{kq}{\sqrt{x^2+y^2+(z+d)^2}}V(x,y,z)=x2+y2+(z−d)2​kq​−x2+y2+(z+d)2​kq​

E2 介質中的電磁場

電介質與電位移向量

介質極化 → 束縛電荷 ρb=−∇⋅P\rho_b = -\nabla\cdot\mathbf{P}ρb​=−∇⋅P。

電位移向量：

D=ε0E+P=εE\mathbf{D} = \varepsilon_0\mathbf{E} + \mathbf{P} = \varepsilon\mathbf{E}D=ε0​E+P=εE

Gauss 定律在介質中：∇⋅D=ρf\nabla\cdot\mathbf{D} = \rho_f∇⋅D=ρf​（僅自由電荷）。

磁性材料與磁場強度

磁化 → 束縛電流 Jb=∇×M\mathbf{J}_b = \nabla\times\mathbf{M}Jb​=∇×M。

磁場強度：

H=Bμ0−M\mathbf{H} = \frac{\mathbf{B}}{\mu_0} - \mathbf{M}H=μ0​B​−M

Ampère 定律在介質中：∇×H=Jf\nabla\times\mathbf{H} = \mathbf{J}_f∇×H=Jf​（僅自由電流）。

E3 磁向量勢與規範

磁向量勢 A\mathbf{A}A

由 ∇⋅B=0\nabla\cdot\mathbf{B} = 0∇⋅B=0 可引入磁向量勢：

B=∇×A\mathbf{B} = \nabla\times\mathbf{A}B=∇×A

電磁位與規範變換

電磁位 (A,V)(\mathbf{A}, V)(A,V) 與場的關係：

B=∇×A,E=−∇V−∂A∂t\mathbf{B} = \nabla\times\mathbf{A},\quad \mathbf{E} = -\nabla V - \frac{\partial\mathbf{A}}{\partial t}B=∇×A,E=−∇V−∂t∂A​

規範變換 (Gauge Transformation)：

A′=A+∇χ,V′=V−∂χ∂t\mathbf{A}' = \mathbf{A} + \nabla\chi,\quad V' = V - \frac{\partial\chi}{\partial t}A′=A+∇χ,V′=V−∂t∂χ​

電磁場 (E,B)(\mathbf{E}, \mathbf{B})(E,B) 在規範變換下不變。

常用規範：

  • Coulomb 規範：∇⋅A=0\nabla\cdot\mathbf{A} = 0∇⋅A=0（簡化靜磁問題）
  • Lorenz 規範：∇⋅A+1c2∂V∂t=0\nabla\cdot\mathbf{A} + \frac{1}{c^2}\frac{\partial V}{\partial t} = 0∇⋅A+c21​∂t∂V​=0（相對論協變）

E4 交流電路與電磁波

複數阻抗 (Phasor Impedance)

交流電路中，電壓和電流用複數表示：

V~=I~Z\tilde{V} = \tilde{I} ZV~=I~Z

阻抗：

  • 電阻：ZR=RZ_R = RZR​=R
  • 電感：ZL=jωLZ_L = j\omega LZL​=jωL（電壓領先電流 90∘90^\circ90∘）
  • 電容：ZC=1jωC=−jωCZ_C = \frac{1}{j\omega C} = -\frac{j}{\omega C}ZC​=jωC1​=−ωCj​（電壓落後電流 90∘90^\circ90∘）

RLC 串聯：Z=R+j(ωL−1ωC)Z = R + j(\omega L - \frac{1}{\omega C})Z=R+j(ωL−ωC1​) 諧振頻率：ω0=1/LC\omega_0 = 1/\sqrt{LC}ω0​=1/LC​
