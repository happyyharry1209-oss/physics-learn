波動與光學

前言

波動是能量傳遞的一種形式。光既是波也是粒子——這種雙重性開啟了近代物理的大門，從雷射到量子力學都源於此。

波的性質

正弦波方程

一維簡諧波沿 xxx 方向傳播：

y(x,t)=Asin⁡(kx−ωt+ϕ)y(x,t) = A\sin(kx - \omega t + \phi)y(x,t)=Asin(kx−ωt+ϕ)
  • AAA：振幅
  • k=2π/λk = 2\pi/\lambdak=2π/λ：波數
  • ω=2πf\omega = 2\pi fω=2πf：角頻率
  • ϕ\phiϕ：相位常數

波速（由波動方程導出）：

v=dxdt=ωk=fλv = \frac{dx}{dt} = \frac{\omega}{k} = f\lambdav=dtdx​=kω​=fλ

繩波波速

波在張緊繩上的傳播速度：

v=Tμv = \sqrt{\frac{T}{\mu}}v=μT​​
  • TTT：繩張力 (N)
  • μ\muμ：繩的線密度 (kg/m)

疊加原理與干涉

兩波相遇時位移相加——這是疊加原理。

條件

  • 建設性干涉（亮紋）：路徑差 =nλ= n\lambda=nλ，n=0,±1,±2,…n=0,\pm1,\pm2,\dotsn=0,±1,±2,…
  • 破壞性干涉（暗紋）：路徑差 =(n+12)λ= (n+\frac12)\lambda=(n+21​)λ

楊氏雙狹縫干涉

亮紋間距：

Δy=λLd\Delta y = \frac{\lambda L}{d}Δy=dλL​

干涉強度分佈（ddd 為狹縫間距）：

I(θ)=I0cos⁡2(πdsin⁡θλ)I(\theta) = I_0 \cos^2\left(\frac{\pi d \sin\theta}{\lambda}\right)I(θ)=I0​cos2(λπdsinθ​)

相干性 (Coherence)：兩束光頻率相同、相位差恆定才能產生穩定干涉條紋。

薄膜干涉 (Thin-Film Interference)

光從折射率 nnn 的薄膜上下表面反射時產生干涉：

光程差（垂直入射）：

2nt={(m+12)λ建設性（半波損失奇數次）mλ破壞性（半波損失偶數次）2 n t = \begin{cases} (m+\frac12)\lambda & \text{建設性（半波損失奇數次）} \\ m\lambda & \text{破壞性（半波損失偶數次）} \end{cases}2nt={(m+21​)λmλ​建設性（半波損失奇數次）破壞性（半波損失偶數次）​

半波損失 (Phase Change on Reflection)：光從折射率小的介質射向折射率大的介質時，反射光相位反轉（π\piπ 相位跳變）。

應用：肥皂泡彩色條紋、抗反射鍍膜（相消干涉消除反射）。

繞射 (Diffraction)

波通過狹縫或障礙物時會彎曲擴散。

單狹縫繞射

暗紋條件：asin⁡θ=nλa\sin\theta = n\lambdaasinθ=nλ，n=±1,±2,…n = \pm1, \pm2, \dotsn=±1,±2,…

中央亮帶寬度：

Δy0=2λLa\Delta y_0 = \frac{2\lambda L}{a}Δy0​=a2λL​

中央亮帶寬度為其他亮帶的兩倍。

I(θ)=I0(sin⁡αα)2,α=πasin⁡θλI(\theta) = I_0 \left(\frac{\sin\alpha}{\alpha}\right)^2,\quad \alpha = \frac{\pi a\sin\theta}{\lambda}I(θ)=I0​(αsinα​)2,α=λπasinθ​

光電效應

光照射金屬表面使電子逸出，愛因斯坦提出光子概念解釋：

hf=ϕ+Kmax⁡hf = \phi + K_{\max}hf=ϕ+Kmax​
  • h=6.63×10−34h = 6.63 \times 10^{-34}h=6.63×10−34 J·s（普朗克常數）
  • ϕ\phiϕ：功函數（金屬的束縛能）
  • Kmax⁡K_{\max}Kmax​：逸出電子的最大動能

截止電壓 (Stopping Potential)

利用反向電壓 VsV_sVs​ 使最大動能的電子恰好無法到達陽極：

eVs=Kmax⁡=hf−ϕeV_s = K_{\max} = hf - \phieVs​=Kmax​=hf−ϕ

VsV_sVs​-fff 圖分析：

  • 斜率 = h/eh/eh/e（可測定普朗克常數）
  • 與 fff 軸交點 = 底限頻率 f0=ϕ/hf_0 = \phi/hf0​=ϕ/h

重要結論：

  • 光的頻率 <<< 底限頻率 → 無論光多強都不產生光電效應
  • 光強度只影響電子數量，不影響電子動能

物質波（德布羅意波）

德布羅意提出：所有物質都具有波粒二象性。

德布羅意波長：

λ=hp=hmv\lambda = \frac{h}{p} = \frac{h}{mv}λ=ph​=mvh​

電子波長（經電壓 VVV 加速後）：

λ=h2meeV=h2meK\lambda = \frac{h}{\sqrt{2m_e eV}} = \frac{h}{\sqrt{2m_e K}}λ=2me​eV​h​=2me​K​h​

質量越大的物體物質波長越短，波動性越不明顯。
