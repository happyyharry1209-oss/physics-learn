磁學與電磁感應（進階）

磁場與磁通量定量定義、長直導線與螺線管磁場公式、安培力與 Lorentz 力、帶電粒子圓周運動、Faraday 定律、發電機原理。

前言

磁力是自然界的基本力之一。從指南針到 MRI，從電動機到粒子加速器——磁學是現代科技的核心。沒有磁學就沒有發電廠、沒有馬達、也沒有現代文明。

磁鐵與磁場

磁鐵的基本性質

  • 磁鐵有兩個極：N 極（北極）和 S 極（南極）
  • 同名極相斥，異名極相吸
  • 磁鐵能吸引鐵 (Fe)、鈷 (Co)、鎳 (Ni) 等鐵磁性物質
  • 磁鐵無法吸引鋁、銅、塑膠

磁場與磁力線

磁場 B⃗\vec{B}B 的單位為特斯拉 (T)。

  • 磁力線從 N 極出發，進入 S 極（外部）
  • 磁力線是封閉曲線，永遠不相交
  • 磁力線越密，磁場越強

磁通量 (Magnetic Flux)

磁通量是通過某一面積的磁力線總數量的量度：

ΦB=BAcos⁡θ\Phi_B = B A \cos\thetaΦB​=BAcosθ
  • ΦB\Phi_BΦB​：磁通量，單位韋伯 (Wb)，1 Wb=1 T⋅m21\ \text{Wb} = 1\ \text{T}·\text{m}^21 Wb=1 T⋅m2
  • BBB：磁場強度 (T)
  • AAA：面積 (m²)
  • θ\thetaθ：磁場方向與平面法向量（垂直於平面的方向）的夾角

特殊情況：

  • θ=0∘\theta = 0^\circθ=0∘（磁場垂直穿入平面）：ΦB=BA\Phi_B = BAΦB​=BA（最大）
  • θ=90∘\theta = 90^\circθ=90∘（磁場平行於平面）：ΦB=0\Phi_B = 0ΦB​=0
  • θ=180∘\theta = 180^\circθ=180∘（磁場垂直穿出平面）：ΦB=−BA\Phi_B = -BAΦB​=−BA

地磁

地球是一個大磁鐵：指南針 N 極指向地理北極（實際上是地磁南極）。地磁保護我們免受太陽風侵害。

💡 極光：太陽風帶電粒子沿地磁力線進入兩極大氣層，與空氣分子碰撞發出絢麗光芒。

電流的磁效應

厄斯特實驗（1820）

導線通電後旁邊的磁針會偏轉——證明電可以產生磁。

長直導線的磁場

電流 III 在距離 rrr 處產生的磁場強度：

B=μ0I2πrB = \frac{\mu_0 I}{2\pi r}B=2πrμ0​I​
  • μ0=4π×10−7 T⋅m/A\mu_0 = 4\pi \times 10^{-7}\ \text{T·m/A}μ0​=4π×10−7 T⋅m/A（真空磁導率）
  • III：電流 (A)
  • rrr：與導線的垂直距離 (m)

特性：磁場強度與距離成反比，離導線越遠磁場越弱。

螺線管的磁場

螺線管（Solenoid）內部近似均勻磁場：

B=μ0nI=μ0NLIB = \mu_0 n I = \mu_0 \frac{N}{L} IB=μ0​nI=μ0​LN​I
  • n=N/Ln = N/Ln=N/L：單位長度的匝數 (匝/m)
  • NNN：總匝數
  • LLL：螺線管長度 (m)

安培右手定則：右手握住導線，拇指指向電流方向，四指彎曲方向即為磁場方向。

電磁鐵

電磁鐵 = 螺線管 + 鐵芯

特性電磁鐵永久磁鐵
磁性的有無可控制（通電有、斷電無）固定
磁性的強弱可調整固定
磁極方向可改變固定

應用：電磁起重機、電鈴、繼電器、磁浮列車。

帶電粒子在磁場中的運動（Lorentz 力）

運動的帶電粒子在磁場中會受到磁力（Lorentz 力）：

F=qvBsin⁡θF = qvB\sin\thetaF=qvBsinθ
  • qqq：粒子電荷量 (C)
  • vvv：粒子速度 (m/s)
  • BBB：磁場強度 (T)
  • θ\thetaθ：速度方向與磁場方向的夾角

垂直入射 → 等速圓周運動

當 θ=90∘\theta = 90^\circθ=90∘（粒子垂直進入均勻磁場）時，Lorentz 力提供向心力：

qvB=mv2rqvB = \frac{mv^2}{r}qvB=rmv2​

迴旋半徑（Cyclotron Radius）：

r=mvqBr = \frac{mv}{qB}r=qBmv​

迴旋週期：

T=2πmqBT = \frac{2\pi m}{qB}T=qB2πm​

關鍵性質：迴旋週期 TTT 與粒子速度 vvv 和半徑 rrr 無關！這是迴旋加速器的基本原理。

導線在磁場中的受力（安培力）

載流導線在磁場中受到的磁力：

F=BILsin⁡θF = BIL\sin\thetaF=BILsinθ
  • FFF：安培力 (N)
  • BBB：磁場強度 (T)
  • III：電流 (A)
  • LLL：導線長度 (m)
  • θ\thetaθ：電流方向與磁場方向的夾角

佛來明左手定則（判斷受力方向）：

  • 食指 = 磁場方向（N→S）
  • 中指 = 電流方向
  • 拇指 = 受力方向

電動機

電動機將電能轉化為機械能。原理：通電線圈在磁場中受力旋轉。

主要部件：磁鐵、線圈、換向器、電刷。應用於風扇、電動車、洗衣機等。

電磁感應

法拉第定律

感應電動勢的大小與磁通量變化率成正比：

E=−NΔΦBΔt\mathcal{E} = -N\frac{\Delta\Phi_B}{\Delta t}E=−NΔtΔΦB​​

冷次定律

感應電流的方向總是抵抗磁通量的變化（公式中的負號）。

佛來明右手定則（判斷感應電流方向）：

  • 食指 = 磁場方向
  • 拇指 = 導體運動方向
  • 中指 = 感應電流方向

發電機

發電機將機械能轉化為電能（電動機的逆過程）。線圈在磁場中旋轉產生感應電動勢。

發電方式：火力、水力、風力、核能 → 渦輪 → 發電機
